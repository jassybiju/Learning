# Learning

## Prompt for ChatGPT

You are a senior software engineer, system designer, and interviewer.

I am following the Brototype self-learning program.
Each module lasts 8 days and ends with an exam, but my goal is
professional-level mastery, not just passing.

Module Name: <MODULE_NAME>
Module Tasks / Topics Covered:
<MODULE_TASKS>

Your task:
Create an ORDERED, PROGRESSIVE list of questions that I must be able to
answer confidently to claim PROFESSIONAL DEVELOPER LEVEL mastery of this module.

Rules:
1. Questions must be ordered from fundamentals → internals → advanced → production → interview level.
2. Cover:
   - Core concepts
   - Internal working / how it works under the hood
   - Real-world usage
   - Performance implications
   - Common mistakes & failure scenarios
   - Debugging & optimization
   - System design & architectural trade-offs
3. Do NOT include beginner or shallow questions.
4. Assume I already know basic syntax.
5. Questions should be deep enough that:
   - They can be asked in real interviews
   - They are useful in real production systems
6. Output should be grouped into logical PHASES with clear titles.
7. Target output size: 100–130 questions.

Tone:
Serious, engineering-focused, no fluff, no motivation talk.

End the response with:
- A short “Professional Benchmark” explaining what mastery looks like for this module.
